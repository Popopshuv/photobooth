# AGENTS.md

## This is NOT the Next.js you know

This starter pins **Next.js 16** with App Router, React 19, and Tailwind v4. APIs, conventions, and file structure may differ from your training data. Read the relevant guide in `node_modules/next/dist/docs/` before writing routing, metadata, or server-component code. Heed deprecation notices.

## Read before working

The full design system rules live in **[CLAUDE.md](./CLAUDE.md)**. The long-form design reference is in **[GROUP-D-SYSTEM.md](./GROUP-D-SYSTEM.md)**. Both files are load-bearing — read them before making non-trivial design or motion decisions.

The bar for this codebase: **minimal sites with elevated micro-interactions.** Strip the surface to almost nothing, then layer cinematic motion on top. If a section doesn't move, drift, mask in, or react, it's probably wrong.
